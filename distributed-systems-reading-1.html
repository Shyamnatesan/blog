Introduction

Apps are either stateless or stateful.

  • Stateless — Web servers → just add more nodes
  • Stateful — Databases, caches → needs coordination, not just nodes

Distributed systems = making multiple stateful nodes behave as one unified system.

Distributing data means at least one of:

  • Partitioning — split data across nodes
  • Replication — copy data across nodes
